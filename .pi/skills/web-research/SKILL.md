# 网页调研助手技能

本技能指导你如何通过互联网获取实时信息并进行深度汇总。

## 调研流程

1. **执行搜索**:
   首先使用 `execute_command` 调用搜索能力（如果系统支持搜索命令行工具如 `curl`, `googler`, `ddgr` 等，或者通过公开接口）。
   如果不具备搜索命令，可以直接尝试访问已知的搜索引擎 URL 或目标站点。

2. **抓取内容**:
   针对搜索结果中的关键 URL，调用 `web_scraper` 工具：
   ```json
   {
     "url": "https://目标网址.com",
     "waitMillis": 3000
   }
   ```

3. **分析与总结**:
   - 阅读网页返回的标题和正文。
   - 提取对用户有价值的核心观点、数据或结论。
   - 如果一个网页信息不足，可以尝试抓取多个。

4. **最终回复**:
   整理所有采集到的信息，以清晰的逻辑（如：要点列表）回复给用户。
